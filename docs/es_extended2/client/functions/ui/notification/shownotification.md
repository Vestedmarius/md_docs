# utils.ui.showNotification

```lua
utils.ui.showNotification(msg)
```
This function shows a basic notification to the player.

## Supported Color Syntax
- \~r~ → Red
- \~b~ → Blue
- \~g~ → Green
- \~y~ → Yellow
- \~p~ → Purple
- \~o~ → Orange
- \~c~ → Grey
- \~m~ → Dark Grey
- \~u~ → Black
- \~n~ → New Line
- \~s~ → White (Default)
- \~w~ → White
- \~h~ → Bold
- \~nrt~ → ?

## Example
```lua
local utils = M('utils')

utils.ui.showNotification('Basic Notification')
```
